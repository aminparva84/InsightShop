Brand logos for the homepage carousel
=====================================

Place logo image files here to use them in the "Brands we work with" carousel
under Featured Products. If a file is missing, the app will use CDN fallbacks
where available.

Expected filenames (matching backend brands in models/product.py BRAND_CHOICES):
  ralph_lauren.png   - Ralph Lauren
  costa.png          - Costa
  champion.png       - Champion
  blueberry.png      - Blueberry
  polo.png           - Polo
  tnf.png            - The North Face
  levis.png          - Levi's
  tommy.png          - Tommy Hilfiger
  wrangler.png       - Wrangler

Recommended: PNG or SVG, transparent background, height ~64–128px (width auto).
The carousel scales logos to 32px height (26px on tablet, 22px on mobile).
