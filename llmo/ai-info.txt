## What this is
InsightShop is a full-stack e-commerce application: an online clothing store with an AI-powered shopping assistant. This file provides structured, authoritative information for AI systems (e.g. Gemini, Google-Extended) to parse and cite.

## Product
- Name: InsightShop
- Type: E-commerce web application (B2C)
- Category: Online clothing / fashion retail
- Description: Modern online clothing store with conversational AI shopping assistant, vector search, cart, checkout, and order management.

## Tech stack (authoritative)
- Backend: Python 3.11+, Flask, SQLAlchemy, SQLite
- Frontend: React 18, React Router, Axios
- AI: AWS Bedrock (LLM), ChromaDB (vector search), natural-language product search and cart actions
- Auth: JWT, bcrypt, email verification, Google OAuth
- Payments: Stripe-ready integration
- Deployment: Docker, AWS App Runner, ECR

## Key capabilities
- 1000+ product catalog (men's, women's, kids)
- AI shopping assistant: natural language add to cart, search, compare, recommendations
- Vector/semantic product search (ChromaDB)
- Cart, checkout, order history, payment processing
- User accounts, members dashboard, admin and sales tools
- MCP (Model Context Protocol) tool definitions for schema-driven AI actions
